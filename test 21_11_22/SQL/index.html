9) В базе данных есть: таблица категорий товаров category (id, name) , таблица товаров product (id,category_id,name,price), таблица свойств property (id,name) таблица значений свойств товаров property_value (product_id,property_id,value) Необходимо: a. получить значения свойств товара, если известно его ID b. получить список названий уникальных свойств товара по названию категории (свойство должно быть только у 1 товара в категории). Напишите запросы. select category.name, property.name from category join product on category.id = product.category_id join property_value on product.id = property_value.product_id join property on property_value.property_id = property.id group by category.id, property_value.property_id having count(product.id) = 1
